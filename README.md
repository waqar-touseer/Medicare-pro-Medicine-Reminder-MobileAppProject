# Medicare-pro-Medicine-Reminder-MobileAppProject
The MedicineReminder Android App is a simple and intuitive mobile application designed to help users manage their medication schedules. This app allows users to set reminders for taking their medications at specified times, ensuring they never miss a dose. It is particularly useful for individuals with chronic conditions or those who require multiple medications throughout the day.

The app provides a user-friendly interface to add, edit, and delete medication entries, along with push notifications to remind users when it’s time to take their medicine. It also stores medicine schedules locally on the device, ensuring data persistence even when offline.

Features
Medicine Scheduling: Set custom reminders for each medication, including name, dosage, frequency, and time.
Push Notifications: Receive timely reminders to take medications at the scheduled times.
SQLite Database: Store medication details and schedules locally for offline access.
User-Friendly Interface: Easy-to-use interface with clear options for managing and tracking medicines.
Editable Entries: Add, edit, and delete medicine entries as needed.
